# STKI

Repository ini berisi kumpulan tugas dan praktikum mata kuliah  
**Sistem Temu Kembali Informasi (STKI)**.

## Deskripsi
Mata kuliah Sistem Temu Kembali Informasi membahas konsep, metode, dan implementasi
sistem pencarian informasi, pengolahan teks, indexing, serta evaluasi hasil temu
kembali informasi menggunakan pendekatan komputasional.

Repository ini dibuat sebagai dokumentasi dan pengumpulan tugas per minggu.

## Struktur Folder
- **Minggu-2** : Materi dan tugas minggu ke-2
- **Minggu 3** : Materi dan tugas minggu ke-3
- **Minggu 4** : Materi dan tugas minggu ke-4
- **Minggu 8** : Praktikum dan implementasi STKI
- **Minggu 9** : Tugas lanjutan STKI
- **Minggu 10** : Evaluasi dan pengembangan sistem
- **Minggu 11** : Analisis hasil dan kesimpulan

## Tools & Teknologi
- Python
- Jupyter Notebook
- Scikit-learn
- Pandas
- NumPy
